# Shot Types Reference Library

## 景别 (Shot Sizes)

### Extreme Wide Shot (超大全景)
**用途：** 建立环境规模感，渺小感，史诗感
**适合：** 开场建立镜头，旅行，自然，城市纵览
**提示词：** `extreme wide shot, establishing, environment-dominant, subject small in frame`

### Wide Shot (全景)
**用途：** 展示完整人物和周围环境关系
**适合：** 场景介绍，动作全貌，多人互动
**提示词：** `wide shot, full body visible, environment context`

### Medium Wide Shot (中全景 / 美式景别)
**用途：** 膝盖以上，动作清晰，有环境感
**适合：** 对话，走路，手势重要的场景
**提示词：** `medium wide shot, knees up, balanced framing`

### Medium Shot (中景)
**用途：** 腰部以上，最常见的叙事景别
**适合：** 对话，演讲，展示产品使用
**提示词：** `medium shot, waist up, conversational framing`

### Medium Close-up (中近景)
**用途：** 胸部以上，介于中景和近景之间
**适合：** 采访，情绪表达，专注脸部
**提示词：** `medium close-up, chest up, facial expression visible`

### Close-up (近景/特写)
**用途：** 头肩部或局部，强调表情/细节
**适合：** 情绪高潮，产品细节，关键信息
**提示词：** `close-up shot, face filling frame, intimate, detailed`

### Extreme Close-up (大特写)
**用途：** 局部特写，极度放大某个细节
**适合：** 眼睛、手指、产品纹理、小物件
**提示词：** `extreme close-up, macro detail, texture visible, isolated detail`

## 镜头角度 (Camera Angles)

| 角度 | 效果 | 提示词 |
|------|------|--------|
| Eye Level (平视) | 自然、平等 | `eye level angle, neutral perspective` |
| Low Angle (仰拍) | 主体强大、威严 | `low angle shot, looking up at subject, powerful` |
| High Angle (俯拍) | 主体渺小/脆弱，全局感 | `high angle shot, looking down, overview` |
| Bird's Eye (正俯) | 图案感，抽象，上帝视角 | `bird's eye view, directly overhead, top-down` |
| Dutch Angle (斜角) | 紧张感，不稳定，戏剧性 | `dutch angle, tilted frame, tension` |
| POV (主观视角) | 沉浸感，代入感 | `POV shot, first person perspective, immersive` |
| Over the Shoulder (过肩) | 对话感，关系感 | `over-the-shoulder shot, two-person dynamic` |

## 镜头运动 (Camera Movements)

### 无运动类
| 运动 | 说明 | 提示词 |
|------|------|--------|
| Static (静止) | 完全固定，稳定 | `static shot, locked off camera, perfectly stable` |
| Subtle breathing (微动) | 极轻微手持感 | `subtle handheld, organic breathing movement` |

### 平移类
| 运动 | 说明 | 提示词 |
|------|------|--------|
| Pan Left/Right (摇镜) | 横向旋转扫描 | `camera pan left/right, smooth rotation` |
| Tilt Up/Down (俯仰) | 上下旋转 | `tilt up/down, vertical pan` |
| Roll (旋转) | 镜头旋转 | `camera roll, rotating perspective` |

### 移动类
| 运动 | 说明 | 提示词 |
|------|------|--------|
| Push In (推镜) | 慢慢靠近主体，紧张/聚焦 | `slow push-in toward subject, gradual approach` |
| Pull Out (拉镜) | 慢慢远离，揭示感 | `pull back, slowly revealing wider scene` |
| Dolly (移动推拉) | 整体前后移动，立体感强 | `dolly shot, camera physically moving forward` |
| Tracking (跟拍) | 跟随主体横向移动 | `tracking shot following subject, parallel movement` |
| Crane Up/Down (升降) | 上升或下降 | `crane shot rising/descending, vertical movement` |
| Orbit/Arc (环绕) | 围绕主体旋转 | `orbital shot, circling around subject 360 degrees` |
| Handheld (手持) | 自然抖动，真实纪实感 | `handheld camera, natural shake, documentary feel` |
| Steadicam (稳定器) | 流畅跟拍，浮动感 | `steadicam, smooth floating movement, fluid follow` |

### 特殊类
| 运动 | 说明 | 提示词 |
|------|------|--------|
| Drone/Aerial (航拍) | 空中俯视 | `aerial drone shot, flying over, elevated perspective` |
| Whip Pan (甩镜) | 极速横摇，转场用 | `whip pan transition, fast blur, motion cut` |
| Rack Focus (焦点切换) | 前后景焦点转换 | `rack focus, focal point shifting from foreground to background` |
| Zoom (变焦) | 光学推拉，独特拉伸感 | `zoom in/out, optical zoom, lens compression effect` |
| Dolly Zoom (希区柯克变焦) | 推进+拉焦，迷幻/震惊 | `dolly zoom, Hitchcock effect, world distortion` |

## 光线类型 (Lighting)

| 光线 | 说明 | 提示词 |
|------|------|--------|
| Golden Hour | 日出/落后暖黄光 | `golden hour, warm amber sunlight, long shadows` |
| Blue Hour | 黄昏前后冷蓝调 | `blue hour, cool twilight, soft ambient glow` |
| Natural Window Light | 窗边自然散射光 | `soft natural window light, diffused daylight, indoor` |
| Hard Light | 强硬单方向光，戏剧感 | `hard directional light, strong shadows, dramatic contrast` |
| Soft Box | 均匀柔和，专业感 | `soft box lighting, even illumination, studio quality` |
| Backlight / Rim Light | 逆光，人物轮廓发光 | `backlit subject, rim lighting, silhouette glow` |
| Neon / Practical Lights | 霓虹/环境光，氛围感 | `neon lighting, practical lights, colored ambient` |
| Cinematic High Contrast | 高对比电影感 | `high contrast cinematic lighting, deep shadows` |
| Low Key | 暗调，神秘/悬疑 | `low key lighting, predominantly dark, moody shadows` |
| High Key | 亮调，轻盈/积极 | `high key lighting, bright and airy, minimal shadows` |

## 速度控制 (Speed)

| 效果 | 说明 | 提示词 |
|------|------|--------|
| Normal Speed | 正常速度 | `normal speed, real-time` |
| Slow Motion | 慢动作，强调细节 | `slow motion, 120fps, time-stretched, dramatic detail` |
| Extreme Slow Mo | 极致慢动作 | `ultra slow motion, 240fps+, hyper-detailed movement` |
| Time Lapse | 延时，时间流逝感 | `time lapse, accelerated time, fast motion` |
| Speed Ramp | 速度变化，由慢到快 | `speed ramp, variable speed, dynamic pacing change` |
